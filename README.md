# 🍟 Barões da Batata

Bem-vindo ao repositório oficial do **Barões da Batata**, um sistema moderno de pedidos online desenvolvido para oferecer uma experiência rápida, intuitiva e totalmente responsiva.

O projeto foi criado com foco em proporcionar uma plataforma completa para pedidos de batatas, salgados e bebidas, permitindo que os clientes personalizem seus pedidos, realizem pagamentos via Pix e acompanhem todo o processo de forma simples e prática.

Além da experiência do cliente, o sistema conta com recursos para gerenciamento de pedidos, administração de produtos e integração com WhatsApp, tornando o atendimento mais rápido e organizado.

Desenvolvido por **Wendel**.
